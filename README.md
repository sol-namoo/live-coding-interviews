# Live Coding Interview Log

기업 면접에서 받았던 라이브 코딩 문제들을 정리하는 레포입니다.

## 목적

- 실제 면접에서 받았던 라이브 코딩 문제를 다시 복기하고 정리하기
- 회사/유형별로 자주 나오는 패턴 파악하기
- 다음 면접에서 비슷한 문제가 나왔을 때 더 빠르고 안정적으로 풀기

## 구조

- `problems/YYYY-MM-DD-company-n-title/`
  - `README.md`: 문제 설명 + 메타데이터
  - `solution.ts` (또는 `solution.<언어>`): 최종 풀이 코드
  - `notes.md`: 면접 상황, 회고

초기에는 `TypeScript` 기준으로 작성하고, 필요하면 이후에 다른 언어 디렉터리를 추가해도 됩니다.

## 새 문제 추가 방법

1. 디렉터리 생성


   ```md
   (실제 면접에서 들은 설명을 최대한 그대로 복원해서 적기)

   입력: nums = [2, 7, 11, 15], target = 9  
   출력: [0, 1]
   ```
